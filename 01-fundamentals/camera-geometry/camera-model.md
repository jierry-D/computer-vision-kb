# 相机模型

## 针孔相机模型

```
3D 世界坐标 → 相机坐标 → 图像平面坐标 → 像素坐标
     [R|t]           K（内参矩阵）
```

### 内参矩阵 K

```
K = [fx  0  cx]
    [ 0 fy  cy]
    [ 0  0   1]
```

- `fx`, `fy`：焦距（像素单位）
- `cx`, `cy`：主点（光轴与像平面的交点）

### 外参 [R | t]

- R：3×3 旋转矩阵，相机相对世界坐标系的朝向
- t：3×1 平移向量

## 畸变模型

| 类型 | 系数 | 常见镜头 |
|------|------|---------|
| 径向畸变 | k1, k2, k3 | 普通镜头（桶形/枕形） |
| 切向畸变 | p1, p2 | 镜头安装不平行 |
| 鱼眼畸变 | 独立模型 | 超广角鱼眼镜头 |
