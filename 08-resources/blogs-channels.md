### Lilian Weng's Blog（英文）

**关注理由**：Lilian Weng 是 OpenAI 安全团队负责人，她的博客文章是同类中最高质量的综述，通常覆盖一个方向的完整发展脉络。

- "Object Detection in 20 Years: A Survey" — 检测方向必读综述

- "Diffusion Models" — 扩散模型最清晰的技术解析
- "Prompt Engineering" — Prompt 技术完整指南

💡 **Tips**：她的每篇文章都配有大量公式推导和图示，建议结合论文原文一起阅读。

### Papers With Code（英文）

**关注理由**：最重要的论文追踪网站，每篇论文都链接到对应代码，SOTA 榜单实时更新。

**更新频率**：每日更新（跟随 arxiv）

**核心功能**：
- SOTA 榜单：按任务/数据集查看当前最好的方法
- Methods 页面：了解某种技术的发展历史
- Datasets 页面：快速找到某个方向的标准数据集
- Trending 页面：当日热门论文

**使用技巧**：
```
找最新方法：选任务 → 选数据集 → 查 SOTA 表格
找代码复现：论文页 → 点击 "Code" 标签 → 找 Star 多的实现
发现新方向：每天看 Trending，5分钟了解最新进展
```

## RSS 订阅建议

用 RSS 统一管理信息源，避免被算法推荐分散注意力：

**推荐 RSS 阅读器**：
- **Inoreader**（网页/App，免费版够用）
- **Feedly**（界面美观，免费版支持 100 个源）
- **NetNewsWire**（macOS/iOS，免费开源）

**推荐订阅源**：

```
arxiv cs.CV 日报：
  https://export.arxiv.org/rss/cs.CV

Papers With Code 趋势：
  https://paperswithcode.com/latest  （用浏览器插件抓取）

Lilian Weng Blog：
  https://lilianweng.github.io/index.xml

Distill.pub：
  https://distill.pub/rss.xml

Towards Data Science：
  https://towardsdatascience.com/feed

The Gradient：
  https://thegradient.pub/rss/
```

**订阅策略建议**：
- 每天早上 10 分钟浏览标题，标记感兴趣的
- 每周集中精读 2-3 篇
- 每月整理笔记，防止信息过载
- 不要订阅超过 20 个源，质量 > 数量

⚠️ **注意**：arxiv cs.CV 每天有 100-200 篇新论文，不可能全读。建议用 Papers With Code 的 Trending 页代替直接订阅 arxiv，只看被社区关注的高价值论文。
