# 机器视觉知识库

| 目录 | 内容 |

| [01-fundamentals](./01-fundamentals/) | 数学基础、图像处理、相机几何、深度学习、三维视觉 |
| [02-tasks](./02-tasks/) | 按视觉任务分类：检测、分割、跟踪、OCR 等 |
| [03-papers](./03-papers/) | 论文阅读笔记（含模板） |
| [04-datasets](./04-datasets/) | 常用数据集索引与说明 |
| [05-toolchain](./05-toolchain/) | 环境配置、核心库、标注工具、模型部署 |
| [06-demos](./06-demos/) | 自己的学习与实验代码 |
| [07-awesome](./07-awesome/) | 精选开源项目与工具收藏 |
| [08-resources](./08-resources/) | 书籍、课程、博客等学习资源 |
