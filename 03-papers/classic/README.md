# 经典论文清单

> 机器视觉必读经典，建议按年份顺序阅读。

## 图像分类基础

| 论文 | 年份 | 贡献 |
|------|------|------|
| AlexNet（ImageNet Classification with Deep CNNs） | 2012 | 深度学习 CV 起点 |
| VGGNet | 2014 | 统一 3×3 卷积 |
| GoogLeNet/Inception | 2014 | Inception 模块 |
| ResNet（Deep Residual Learning） | 2015 | 残差连接，训练更深网络 |
| DenseNet | 2017 | 密集连接 |
| MobileNetV1/V2/V3 | 2017-2019 | 轻量化 |
| EfficientNet | 2019 | NAS 复合缩放 |

## 目标检测基础

| 论文 | 年份 | 贡献 |
|------|------|------|
| R-CNN | 2014 | 检测+深度学习起点 |
| Fast R-CNN | 2015 | ROI Pooling |
| Faster R-CNN | 2015 | RPN，端到端 |
| YOLO v1 | 2016 | 实时检测 |
| SSD | 2016 | 多尺度单阶段 |
| Feature Pyramid Network（FPN） | 2017 | 多尺度特征融合 |
